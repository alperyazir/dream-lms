## Story

**so that** QA can audit clean, production-ready code without noise from development artifacts.

6. `.gitignore` covers all dev artifacts (`nohup.out`, `todo.txt`, `*.pyc`, `__pycache__`, `.DS_Store`)
7. All code formatted consistently (`black` for Python, `prettier` for TypeScript)
