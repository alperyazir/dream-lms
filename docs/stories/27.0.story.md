# Story 27.0: Codebase Cleanup

## Status

Draft

## Story

**As a** development team,
**I want** to clean up dead code, debug artifacts, and dev-only leftovers from the codebase,
**so that** QA can audit clean, production-ready code without noise from development artifacts.

## Acceptance Criteria

1. Zero `console.log/debug/warn` in production frontend code (keep only intentional `console.error` in catch blocks)
2. Zero TODO/HACK/FIXME comments without linked issues — resolve or remove them
3. Backend linter (`ruff`) passes with zero errors
4. Frontend linter (`eslint`) passes with zero errors
5. No unused dependencies in `package.json` or `pyproject.toml`
6. `.gitignore` covers all dev artifacts (`nohup.out`, `todo.txt`, `*.pyc`, `__pycache__`, `.DS_Store`)
7. All code formatted consistently (`black` for Python, `prettier` for TypeScript)

## Tasks / Subtasks

- [ ] Task 1: Remove frontend console.log/debug/warn calls (AC: 1)
  - [ ] Search all `.ts` and `.tsx` files for `console.log`, `console.debug`, `console.warn`
  - [ ] Remove debug logging (~101 occurrences across 24 files)
  - [ ] Keep intentional `console.error` in catch blocks
  - [ ] Verify build still compiles after removals

- [ ] Task 2: Resolve Python TODO/HACK/FIXME comments (AC: 2)
  - [ ] Review 5 occurrences in: `analytics_service.py`, `models.py`, `ai_generation.py`, `webhooks.py`
  - [ ] Fix the underlying issue or remove the comment if already resolved
  - [ ] If a TODO is genuinely needed, create a GitHub issue and reference it

- [ ] Task 3: Remove dead code and unused imports (AC: 3, 4)
  - [ ] Run `ruff check --fix` on backend to auto-fix unused imports
  - [ ] Run `eslint --fix` on frontend
  - [ ] Manually review any remaining lint errors
  - [ ] Remove commented-out code blocks (not just imports)

- [ ] Task 4: Clean up dependencies (AC: 5)
  - [ ] Backend: Check `pyproject.toml` for packages not imported anywhere in `backend/app/`
  - [ ] Frontend: Run `npx depcheck` or manually audit `package.json` for unused packages
  - [ ] Remove unused dependencies and verify build/tests still pass

- [ ] Task 5: Update .gitignore and clean untracked files (AC: 6)
  - [ ] Add patterns: `nohup.out`, `todo.txt`, `*.pyc`, `__pycache__/`, `.DS_Store`
  - [ ] Remove any tracked files that should be ignored (`git rm --cached` if needed)
  - [ ] Delete local dev artifacts (`todo.txt`, `nohup.out`)

- [ ] Task 6: Run formatters and verify (AC: 7)
  - [ ] Run `black .` on backend
  - [ ] Run `ruff check .` on backend — zero errors
  - [ ] Run `npx prettier --write "src/**/*.{ts,tsx}"` on frontend
  - [ ] Run `npx eslint src/` on frontend — zero errors
  - [ ] Verify both frontend and backend build successfully

## Dev Notes

- Backend: `backend/app/` — FastAPI application, Python 3.12
- Frontend: `frontend/src/` — React + TypeScript + Vite

- Backend formatter: `black` (line length 88)
